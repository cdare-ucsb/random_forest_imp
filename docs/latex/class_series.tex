\doxysubsection{Detailed Description}
\doxylink{class_series}{Series} class, representing a column of data. 

This class represents a \doxylink{class_series}{Series} object, which is a column of data values. The data values can be of type int, double, or string. The \doxylink{class_series}{Series} class provides methods for adding data values, retrieving data values, and performing operations on the data.

\begin{DoxyReturn}{Returns}
double entropy of the set of labels
\end{DoxyReturn}
The entropy of a set of labels is calculated as follows\+:
\begin{DoxyEnumerate}
\item Count the frequency of each unique label in the set.
\item For each unique label, calculate the probability of that label by dividing its frequency by the total number of labels.
\item For each unique label, calculate the term -\/p \texorpdfstring{$\ast$}{*} log2(p), where p is the probability of the label.
\item Sum all the terms calculated in step 3 to get the entropy.
\end{DoxyEnumerate}
