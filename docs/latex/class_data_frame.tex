The entropy of a set of labels is calculated as follows\+:
\begin{DoxyEnumerate}
\item Count the frequency of each unique label in the set.
\item For each unique label, calculate the probability of that label by dividing its frequency by the total number of labels.
\item For each unique label, calculate the term -\/p \texorpdfstring{$\ast$}{*} log2(p), where p is the probability of the label.
\item Sum all the terms calculated in step 3 to get the entropy.
\end{DoxyEnumerate}

Entropy is a measure of the impurity or disorder of the set of labels. A higher entropy value indicates more disorder. \Hypertarget{class_data_frame_a61187bfa789567ccc329a3b363b6929f}\index{DataFrame@{DataFrame}!calculateInformationGain@{calculateInformationGain}}

Information gain is a measure of the reduction in entropy that results from splitting a set of data on a particular attribute. The information gain of an attribute is calculated as follows\+:
\begin{DoxyEnumerate}
\item Calculate the entropy of the set of labels.
\item For each unique value of the attribute, calculate the weighted average of the entropy of the labels after splitting on that value.
\item Calculate the information gain as the entropy of the set of labels minus the weighted average entropy after splitting.
\item Return the information gain.
