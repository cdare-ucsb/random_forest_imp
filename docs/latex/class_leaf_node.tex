\doxysubsection{Detailed Description}
A class that represents a leaf node in a decision tree. 

This class represents a leaf node in a decision tree. A leaf node contains a predicted value for the target variable. The \doxylink{class_leaf_node}{Leaf\+Node} class inherits from the \doxylink{class_node}{Node} class and implements the pure virtual methods defined in the \doxylink{class_node}{Node} class.
